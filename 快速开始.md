## 📝 开始写作

### 创建新文章

```bash
npx hexo new "我的第一篇技术文章"
```

文章会在 `source/_posts/` 目录下生成，使用 Markdown 格式编写。

### 文章示例

```markdown
---
title: 我的第一篇技术文章
date: 2026-01-29
tags:
  - 技术
  - 教程
categories:
  - 编程
cover: https://images.unsplash.com/photo-1555066931-4365d14bab8c

---

<!-- more -->

## 更多内容

这部分只在文章详情页显示...
```

## ⚡ 常用命令

```bash
# 启动服务（推荐）
./start.sh

# 或手动启动
npx hexo server

# 创建文章
npx hexo new "文章标题"

# 清理缓存
npx hexo clean

# 生成静态文件
npx hexo generate

# 查看项目信息
./info.sh
```

## 🚀 部署到线上

### 方式 1: Vercel（推荐）

1. 将代码推送到 GitHub
2. 在 [Vercel](https://vercel.com) 导入项目
3. 构建命令：`npx hexo generate`
4. 输出目录：`public`
5. 一键部署

### 方式 2: GitHub Pages

```bash
# 安装部署插件
npm install hexo-deployer-git --save

# 配置 _config.yml
deploy:
  type: git
  repo: https://github.com/username/username.github.io
  branch: main

# 部署
npx hexo deploy
```

## ❓ 常见问题

**Q: 如何停止服务器？**
A: 在终端按 `Ctrl+C`

**Q: 修改配置后没生效？**
A: 运行 `npx hexo clean && npx hexo generate` 清理缓存

**Q: 如何更换主题？**
A: 修改 `_config.yml` 中的 `theme` 字段

**Q: 如何添加图片？**
A: 将图片放在 `source/images/` 目录，使用 `/images/xxx.jpg` 引用
